# 更新手順 / update

新しいリリースへ安全に更新する方法です

## 更新前にバックアップするもの・上書きしてはいけないもの

- config.php
- SQLiteデータベース（*.dbファイル）
- 投稿画像
- サムネイル
- セッションなど

## `config.example.php`に設定項目が増えた場合の反映方法

新しく増えた設定を末尾等に追加してください

## DB移行スクリプトが必要なバージョン

現在はv2からv3系統へのみです

## 更新後の構文チェックとスモークテスト

`scrips/lint-php.sh`、`scrips/smoke-test.sh`を実行してください

## 不具合時の元バージョンへの戻し方

githubのリリースより旧バージョンをダウンロードしてください
上書きしてはいけないものは上記に準じます
